║                   ⚡ TROUBLESHOOTING                               ║
╚════════════════════════════════════════════════════════════════════╝

If deployment doesn't work:

1. Refresh the page (F5)
2. Check internet connection
3. Try again after 1 minute
4. Clear browser cache
5. Try different browser


If URL doesn't work:

1. Check URL copied completely
2. Make sure it starts with "https://"
3. Try opening in incognito/private mode
4. Wait 2-3 minutes after deployment
5. Share screenshot if URL not ready yet

If you can't deploy right now, use screenshots:

WINDOWS:
  1. Press: Windows + Shift + S
  2. Drag to select area
  3. Paste in email/WhatsApp

MAC:
  1. Press: Cmd + Shift + 4
  2. Drag to select area
  3. Image saved to desktop

FULL PAGE (CHROME):
  1. Press F12
  2. Press Ctrl + Shift + P (or Cmd + Shift + P)
  3. Type "screenshot"
  4. Choose "Capture full size screenshot"
